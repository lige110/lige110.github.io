Vim commands

Posted on 2020-03-02

This Markdown note contain the most command Vim commands.

Vim#

1. insert#

Command Effect
a insert after cursor
A insert at the end of line
i insert before current cursor
I insert at the beginning of line
o insert a new lien below the cursor
O Insert a new line above the cursor

2. save#

2. operation with lines#

1) move cursor#

  1. up down left right

    h, j, k, l left, down, up, right

  2. move to file beginning or file end

    gg file beginning
    G file end

  3. move to line beginning or end

    ^ line beginning
    $ line end

  4. move to certain line

    :n move to n line

2) delete or paste#

  1. delete characters

    x delete one char
    nx delete n chars

  2. delete line

    dd cut one line
    ndd cut n lines
    :n1,n2d cut the lline range from n1 to n2

  3. paste

    p paste below the cursor
    P paste above the curosr

  4. delete all file

    dG delete until file end

3) copy#

yy copy current line
nyy copy n lines

4) undo and redo#

u undo last command
ctrl + r redo last command

5) replace#

r replace one char
R enter replace mod until “esc”

3. Vim settings#

Vim setting file should be stored under ~/.vimrc

command settings

:sest nu/nonu display the index number of line
:syntax on/off display color for grammar highilght
:set list/nolist dispaly char for table or enter

4. search#

/keyword search after cursor
?keyword search above cursor
n to next keyword

switch#

:1,10/old/new/g replace old with new for first 10 lines
:%s/old/new/g replace old with new for whole file

comment several lines

;1,10s/^/#/g comment first 10 lines
:1,10/^#//g cancel comments for first 10 lines
