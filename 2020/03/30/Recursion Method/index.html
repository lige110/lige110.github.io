Problem Solving with Recursion Method

Posted on 2020-03-30 Edited on 2020-04-08

Ideas about how to solve problems recursively.

Recursion Method#

Concept#

Some problems can be divided into smaller problems, and each smaller problem has the same requirements as the original problem, then a recursive method can be considered.

How to implement#

There are some key points to be considered when implementing a recursive method.

  1. Reconstruct the problem.
    Whether this problem can be divided into sub-problem which has the same requirement as the origin problem?
  2. Define the boundary of the problem.
    Since we are always calling the same method, we need to tell the program when to stop, or we will get into an infinite loop.
  3. Think about the body of the method.
    We need to think about, what is always conducted, in order to solve the problem.
  4. Define the return value of the method.
    Consider what to return when the sub-problem is solved.
